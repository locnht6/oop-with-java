# Object Oriented Programming with Java

This repository contains small projects developed during the study of **Object-Oriented Programming (OOP) with Java**.

## 📚 Repository Structure

Projects are organized into `Session` folders, each corresponding to a specific topic.

```text
Object-Oriented-Programming-with-Java/
│
├── Session01/
├── Session02/
├── Session03/
├── ...
└── Session07/
```

## 🎯 Objectives

This repository was created to:

* Learn and practice Object-Oriented Programming principles in Java.
* Store projects organized by learning topics.
* Track learning progress and improve Java programming skills.
* Build a solid foundation for larger Java projects in the future.

## 🛠️ Technologies Used

* Java
* NetBeans
* Git & GitHub

## 📖 Learning Topics

The following topics are covered in this repository:

* Variables & Data Types
* Method

  * Overload
* Object & Class

  * Constructor
  * Access Specifier / Access Modifier
  * How Objects Are Stored in Memory (RAM)
* Static
* Input & Try-Catch
* Array

  * Primitive Array
  * Object Array
* Inheritance

  * Override
  * Memory Organization in Inheritance Hierarchies
  * Abstract Class
  * Anonymous Class
* Interface
* OOP Concepts / OOP Principles / Fundamental Concepts of OOP / Four Pillars of OOP

  * Abstraction
  * Encapsulation
  * Inheritance
  * Polymorphism
* Collections Framework
* Wrapper Class

  * Boxing / Unboxing
  * Immutable Class / Mutable Class
  * equals() & compareTo()
  * Pool
